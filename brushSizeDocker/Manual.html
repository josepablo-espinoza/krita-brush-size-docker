Brush Size Docker for Krita

The Brush Size Docker plugin allows you to quickly and easily adjust the size of your brush in Krita using predefined size presets or by manually setting the size.

Features

  1. Preset Selector: Choose from Small, Medium, Large, or Current Brush presets.
  2. Current Brush preset: It will use the current brush size to generte Size 1 (1/5), Size 2 (1/2) and Size 4 (5/3)
  3. Manual Size Input: Input box to type the exact size.
  4. Size Slider: Slider to adjust the preset

Launching the Docker

  1. Navigate to Settings -> Dockers and enable Brush Size Docker to display the Docker.

Usage

Once the Brush Size Docker is enabled, it will appear as a new docker in Krita.

Notes

Currently theres no available method to store user generated presets
